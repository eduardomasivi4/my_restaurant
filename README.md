My restaurant

Um arquivo que terá:
- apresentação
- menu
- sobre
- formulario de reserva

apresentaçao:
- uma imagem no fundo
- o nome do restaurante
- um frase para usar depois do nome

menu:
- imagem e nome de alguns pratos
- botao ver menu

saber mais:
- uma breve inforamação sobre o restaurante
- um botao saber mais

formulario de reserva
- campos a preencher: nome, email, data, hora, nº de pessoas, observações(alergias, limitações, preferencias, ...).
-  um botao reservar.

ver menu:
- uma barra de navegação com (nome do restaurante, links(home, sobre, reservar))
- um titulo "Menu"
- uma lista de todos os pratos do restaurante.

saber mais e sobre
- uma barra de navegação com (nome do restaurante, links(home, sobre, reservar))
- um titulo "Nossa história"
- uma sessão que conta a historia do restaurante
- missaão, visão, valores do restaurante
- uma parte que fala sobre o chefe de cozinha

para o rodapé:
- informações de contacto
- redes sociais
- localização
